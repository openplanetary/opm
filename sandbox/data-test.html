OpenPlanetaryMap

Read to Dataset

Retrieve data from the dataset. Add coordinates of the point (or use the 'map' button to select a point on the map), then click the 'retrieve' button. Data will appear in the 'Name', 'Description' and 'URL' fields.

Write to Dataset

Add some data to the dataset. Enter details below (use the 'map' button to select a point on the map), and click the 'Submit' button. Enter your CARTO API key in the dialog box when it appears.

OpenPlanetaryMap is a project supported by CARTO through their Grant Program
